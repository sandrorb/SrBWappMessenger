Aplicação de envio automático de mensagens via Whatsapp em fase inicial de implementação.
(Ainda não é um MVP)
A planilha de Excel tem 2 colunas sendo a primeira o nome e a segunda o número de telefone em formato texto. Os dados devem estar em uma aba com nome Dados. A planilha é lida e os resultados são postos na página em uma tabela logo abaixo. Mais abaixo, e ainda desconectada dos dados obtidos da planilha, está o formulário de envio de mensagens. O sistema de envio está projetado para usar os recursos do sistema Twilio.com, Facebook Business com Whatsapp Business. Para isso é necessário ter uma conta Meta Empresarial validada, o que ainda não foi possível. Existem "milhares" de condições de contorno a serem implementadas para o código não quebrar. O código pode ser visto no github: https://github.com/sandrorb/SrBWappMessenger
